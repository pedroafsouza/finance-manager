# Finance Manager

A monorepo for managing personal finances and taxes in Denmark.

## Projects

### [Danish Tax & RSU Manager](./app)

A Next.js application built with Bun to help manage taxes and RSU (Restricted Stock Unit) stock grants in Denmark, particularly for employees of tech companies.

**Features:**
- RSU tracking and vesting schedule management
- Danish tax calculations for stock compensation
- SKAT reporting and documentation generation

**Tech Stack:** Next.js 16, TypeScript, Bun, Tailwind CSS 4

See the [app README](./app/README.md) for more details.

## Getting Started

```bash
# Navigate to the app directory
cd app

# Start the development server
bun dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the application.
